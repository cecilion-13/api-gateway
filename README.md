# Api-gateway-Service

An API Gateway Service for the project "Agriculture Crop System".

Expose endpoints of various Microservices, discover using Eureka Discovery Server.

The List of Microservices are:
  1. Farmer Management
  2. Dealer Management
  3. Accounts Service
  4. Crop Item Management
